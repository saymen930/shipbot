# Ship Bot

Bu bot, gemilerinizi yönetmenize ve takip etmenize yardımcı olur.

## Nasıl Kullanılır

1. Botu sunucunuza ekleyin.
2. `!yardım` komutunu kullanarak mevcut komutları görüntüleyin.
3. Gemilerinizi ekleyin, düzenleyin veya silin.
